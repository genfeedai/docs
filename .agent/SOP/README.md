# Documentation Project - Standard Operating Procedures

This folder contains **docs.genfeed.ai-specific** SOPs.

## Purpose

Document project-specific procedures unique to the documentation site:

- Documentation writing standards
- Docusaurus workflows
- Content organization
- Deployment procedures
- SEO optimization

## Structure

```
SOP/
├── README.md (this file)
├── documentation-writing.md
├── docusaurus-maintenance.md
├── content-organization.md
└── ... (other docs-specific SOPs)
```

## When to Add SOPs Here vs Workspace

### Add Here (Project SOP)

- Docusaurus-specific patterns
- Documentation structure/organization
- Content writing guidelines
- Image/asset management
- Documentation deployment
- SEO practices for docs

### Add to Workspace (`../../.agent/SOP/`)

- General coding standards
- Git workflows
- Code review processes
- Security practices

## Cross-Reference

**Workspace SOPs:** `../../.agent/SOP/` - Apply to all projects  
**Project SOPs:** This folder - Apply only to docs.genfeed.ai

---

**Note:** Start with workspace SOPs. Only create project-specific SOPs when procedures diverge significantly from workspace standards.
