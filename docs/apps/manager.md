# Manager App - Complete Guide

Manager is your content organization hub. Organize ingredients, manage captions, create custom trainings, and perform bulk operations on your content library.

## Overview

Manager helps you organize, manage, and optimize your content library. It's where you build your reusable content assets (ingredients), manage captions, train custom AI models, and perform bulk operations.

## Getting Started

### Accessing Manager

1. Navigate to Manager from main navigation
2. Or use command palette (`⌘K`) and type "manager"
3. Or use keyboard shortcut `⌘2` (Mac) or `Ctrl+2` (Windows/Linux)

### Manager Interface

- **Ingredients Library**: Browse and organize your content
- **Caption Management**: Create and manage video captions
- **Trainings**: Custom AI model training
- **Folders**: Organize content into collections
- **Quick Actions**: Bulk operations and shortcuts

## Ingredients Library

### What Are Ingredients?

Ingredients are reusable content building blocks:

- Generated videos, images, voices, music
- Saved for reuse across projects
- Organized with folders and tags
- Searchable and filterable

### Organizing Ingredients

**Folders:**

- Create folders to organize content
- Nest folders for complex organization
- Drag and drop to move content
- Use folders for projects, campaigns, or content types

**Tags:**

- Add tags to ingredients for easy searching
- Use consistent tagging conventions
- Filter by tags in search
- Bulk tag multiple items

**Collections:**

- Create collections for specific purposes
- Group related ingredients together
- Share collections with team members
- Use collections as templates

### Searching and Filtering

**Search:**

- Search by name, tags, or content
- Filter by content type (video, image, etc.)
- Filter by date created
- Filter by folder or collection

**Filters:**

- Content type (video, image, voice, music, article)
- Date range
- Tags
- Folders
- Model used
- Generation date

### Bulk Operations

**Select Multiple Items:**

- Click checkbox to select individual items
- Select all on page
- Select by filter criteria

**Bulk Actions:**

- **Move to Folder**: Organize multiple items at once
- **Add Tags**: Tag multiple items simultaneously
- **Delete**: Remove multiple items
- **Export**: Download multiple items
- **Publish**: Schedule multiple items for publishing

## Caption Management

### Creating Captions

**Automatic Caption Generation:**

1. Select a video ingredient
2. Click "Generate Captions"
3. Choose language and style
4. Review and edit generated captions
5. Save captions to video

**Manual Caption Creation:**

1. Select video
2. Click "Add Captions"
3. Enter caption text
4. Set timing for each caption
5. Style captions (font, color, position)
6. Save captions

### Caption Styles

- **Font**: Choose from available fonts
- **Color**: Set text and background colors
- **Position**: Top, bottom, or custom position
- **Animation**: Fade, slide, or typewriter effects
- **Size**: Adjustable text size

### Caption Templates

- Save caption styles as templates
- Apply templates to multiple videos
- Create brand-consistent captions
- Quick apply to new content

## Custom Trainings

### What Are Trainings?

Trainings are custom AI model configurations that learn your brand voice, style, and preferences for more consistent content generation.

### Creating a Training

**Step 1: Prepare Training Data**

- Gather examples of your desired style
- Include 10-20 high-quality examples
- Ensure examples represent your brand
- Organize examples by category

**Step 2: Create Training**

1. Go to Trainings section
2. Click "Create New Training"
3. Upload training examples
4. Name your training
5. Add description and tags

**Step 3: Train Model**

1. Select training data
2. Choose training parameters
3. Start training process
4. Monitor training progress
5. Test trained model

**Step 4: Use Training**

- Select training when generating content
- Training influences generation style
- Use for brand-consistent content
- Refine training with more examples

### Training Best Practices

- **Quality Over Quantity**: Better to have 10 great examples than 50 mediocre ones
- **Consistency**: Examples should share similar style
- **Variety**: Include different angles but same style
- **Refinement**: Add more examples over time
- **Testing**: Test training before using in production

## Tips for Effective Management

### Organization Tips

1. **Consistent Naming**: Use clear, descriptive names
2. **Regular Cleanup**: Archive or delete unused content
3. **Tag Everything**: Tags make searching easier
4. **Use Folders**: Don't let library get cluttered
5. **Regular Review**: Periodically review and organize

### Workflow Tips

1. **Save Everything**: Save successful generations as ingredients
2. **Build Library**: Create reusable content library
3. **Use Collections**: Group related content
4. **Train Models**: Invest in custom trainings
5. **Bulk Operations**: Use bulk actions to save time

## Keyboard Shortcuts

- `⌘2` / `Ctrl+2`: Open Manager
- `⌘F` / `Ctrl+F`: Search ingredients
- `⌘N` / `Ctrl+N`: New folder
- `⌘T` / `Ctrl+T`: Add tags
- `Space`: Preview selected item

## Troubleshooting

### Can't Find Content

- Use search with different keywords
- Check filters aren't too restrictive
- Look in different folders
- Search by tags

### Bulk Operations Not Working

- Ensure items are selected
- Check you have permissions
- Verify items aren't locked
- Try smaller batches

### Training Fails

- Check training data quality
- Ensure enough examples
- Verify examples are consistent
- Try different training parameters
