# Dashboard App - Complete Guide

Dashboard provides an overview of your organization, team members, billing, credits, and key metrics all in one place.

## Overview

Dashboard is your organization's command center. View organization analytics, manage team members, track billing and credits, and access key settings and information.

## Getting Started

### Accessing Dashboard

1. Navigate to Dashboard from main navigation
2. Or use command palette (`⌘K`) and type "dashboard"
3. Or use keyboard shortcut `⌘H` (Mac) or `Ctrl+H` (Windows/Linux)

### Dashboard Interface

- **Organization Overview**: Key metrics and stats
- **Member Management**: Team member administration
- **Billing & Credits**: Subscription and credit management
- **Settings**: Organization settings
- **Activity Feed**: Recent organization activity

## Member Management

### Viewing Members

- See all organization members
- View member roles
- See member activity
- Check member status

### Adding Members

**Step 1: Invite Member**

1. Click "Add Member"
2. Enter email address
3. Select role
4. Set permissions
5. Send invitation

**Step 2: Member Accepts**

- Member receives invitation
- Accepts invitation
- Joins organization
- Gains assigned permissions

### Member Roles

**Owner:**

- Full organization control
- Billing management
- Member management
- All permissions

**Admin:**

- Content management
- Member management (limited)
- Settings access
- Analytics access

**Member:**

- Content creation
- Content management
- Limited settings
- View analytics

**Viewer:**

- View content only
- Read-only access
- No editing permissions

### Managing Members

**Actions:**

- Change member role

- Manage permissions

## Keyboard Shortcuts

- `⌘H` / `Ctrl+H`: Open Dashboard
- `⌘M` / `Ctrl+M`: Member management
- `⌘$` / `Ctrl+$`: Billing & credits
- `⌘,` / `Ctrl+,`: Settings
- `Space`: View detailed metrics
