# Studio App - Complete Guide

Studio is your AI-powered content creation workspace. Generate videos, images, articles, voice, and music all in one place with real-time preview and editing.

## Overview

Studio is GenFeed's core content generation app where you create all your AI-powered content. It features a conversational interface, multi-modal generation, and real-time preview capabilities.

## Getting Started

### Accessing Studio

1. Navigate to Studio from the main navigation
2. Or use the command palette (`⌘K` or `Ctrl+K`) and type "studio"
3. Or use keyboard shortcut `⌘1` (Mac) or `Ctrl+1` (Windows/Linux)

### Studio Interface

- **Prompt Bar**: Enter your generation prompts here
- **Model Selector**: Choose which AI model to use
- **Generation History**: View your recent generations
- **Preview Panel**: See your content as it generates
- **Settings**: Configure generation parameters

## Content Generation

### Video Generation

**Workflow:**

1. Click "Generate Video" or type your prompt in the prompt bar
2. Select a video model (Kling AI, Google Veo, Runway, etc.)
3. Enter your video prompt (see [Video Prompting Guide](../video-prompting-mastery.md))
4. Configure settings (duration, aspect ratio, style)
5. Click "Generate"
6. Watch real-time preview as video generates
7. Edit or regenerate if needed

**Best Practices:**

- Be specific about scene, camera movement, and style
- Use the 7-element framework (scene, style, camera, lighting, subject, duration, audio)
- Start with shorter videos (5-10 seconds) for faster results
- Use model-specific prompting techniques

**Models Available:**

- **Kling AI V2**: Creative, artistic videos with fluid motion
- **Google Veo 3**: Cinematic quality, high resolution
- **Runway**: Realistic video generation
- **Luma**: High-quality video output

### Image Generation

**Workflow:**

1. Click "Generate Image" or type image prompt
2. Select image model (Imagen 4, Leonardo AI, SDXL, etc.)
3. Enter detailed image prompt
4. Configure dimensions and style
5. Generate and preview
6. Create variations if needed

**Best Practices:**

- Include subject, style, environment, lighting, composition
- Specify aspect ratio for social media formats
- Use model-specific techniques (see [Model Selection Guide](../prompting/model-selection.md))
- Create multiple variations to find the best result

**Models Available:**

- **Google Imagen 4**: Photorealistic images
- **Leonardo AI**: Artistic and stylized content
- **SDXL**: Versatile image generation
- **DALL-E 3**: Complex concepts and illustrations

### Article Generation

**Workflow:**

1. Click "Generate Article" or type article prompt
2. Select language model (GPT-4, Claude, etc.)
3. Enter article topic and requirements
4. Configure SEO settings and keywords
5. Generate article
6. Edit and refine content
7. Export or publish

**Best Practices:**

- Specify target audience and word count
- Include SEO keywords naturally
- Request specific structure (headings, sections)
- Use tone and style guidelines
- Request multiple versions for A/B testing

**Prompt Template:**

```
Write a [word count]-word article about [topic].
Target audience: [describe audience]
Tone: [professional/casual/friendly]
Structure: Introduction, [X] main points, conclusion
SEO keywords: [keyword1, keyword2, keyword3]
Include: [specific requirements]
```

### Voice Generation

**Workflow:**

1. Click "Generate Voice" or select voice option
2. Choose voice model and voice style
3. Enter text to convert to speech
4. Configure voice settings (speed, pitch, emotion)
5. Generate audio
6. Preview and download

**Best Practices:**

- Use punctuation for natural pauses
- Break long text into sentences
- Match voice style to content tone
- Test different voices for your brand

### Music Generation

**Workflow:**

1. Click "Generate Music"
2. Select music model (MusicGen, etc.)
3. Describe desired music (genre, mood, tempo)
4. Configure duration and style
5. Generate music track
6. Preview and download

**Best Practices:**

- Be specific about genre and mood
- Include tempo preferences (BPM)
- Specify instruments if important
- Describe the emotional feel

## Tips for Best Results

### Prompting Tips

1. **Be Specific**: More detail = better results
2. **Use Examples**: Reference styles you like
3. **Iterate**: Refine prompts based on results
4. **Experiment**: Try different models and settings
5. **Learn from Success**: Save prompts that work well

### Model Selection

- **Speed vs Quality**: Faster models for quick iterations, quality models for final content
- **Style Match**: Choose models that match your desired style
- **Content Type**: Different models excel at different content types
- See [Model Selection Guide](../prompting/model-selection.md) for details

### Workflow Optimization

1. Start with quick iterations
2. Refine prompts based on results
3. Use batch generation for variations
4. Save successful prompts as templates
5. Build a library of reusable content

- `⌘K` / `Ctrl+K`: Open command palette
- `⌘⇧V` / `Ctrl+Shift+V`: Generate video

- `⌘⇧A` / `Ctrl+Shift+A`: Generate article
- `⌘F` / `Ctrl+F`: Search content library

## Troubleshooting

### Generation Fails

- Check your prompt length and clarity
- Try a different model
- Verify you have credits available
- Check internet connection

### Results Don't Match Expectations

- Refine your prompt with more detail
- Try different models
- Adjust generation settings
- Use prompt templates as starting points

### Slow Generation

- Some models are slower than others
- Longer content takes more time
- Check your internet connection
- Try simpler prompts for faster results
