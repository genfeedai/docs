# TikTok Setup

Connect your TikTok account to publish videos directly from GenFeed.ai and grow your audience.

## Prerequisites

- Active TikTok account
- Account in good standing (no violations)
- TikTok app for initial setup
- Email verified on TikTok

## Account Connection

### Connecting to GenFeed.ai

1. Navigate to **Accounts** → **Add Account**
2. Select **TikTok**
3. Click **Connect TikTok Account**
4. Log in with your TikTok credentials
5. Review and accept permissions:
   - Video upload access
   - Profile information
   - Analytics viewing
   - Comment management

### Business Account Benefits

- Advanced analytics
- Commercial music library
- Advertising capabilities
- Link in bio (1000+ followers)

To switch:

1. TikTok app → Profile → Settings
2. Manage Account → Switch to Business
3. Select your category
4. Complete business info

## Content Specifications

### Video Requirements

- **Aspect Ratio**: 9:16 (vertical required)
- **Resolution**:
  - Minimum: 720x1280
  - Recommended: 1080x1920
  - Maximum: 1440x2560
- **Duration**:
  - Standard: 15 seconds to 10 minutes
  - Stories: 15 seconds max
- **File Size**: Up to 287MB
- **Formats**: MP4, MOV, MPEG, AVI

### Audio Specifications

- **Sample Rate**: 44.1kHz or 48kHz
- **Bit Rate**: 128kbps minimum
- **Format**: AAC or MP3
- **Commercial Music**: Available with business account

## Publishing Features

### Video Upload Process

1. Select video from GenFeed.ai library
2. Choose **Publish to TikTok**
3. Add description (up to 2200 characters)
4. Select cover image
5. Add hashtags and mentions
6. Choose privacy settings
7. Schedule or post immediately

### Content Optimization

- **Auto-Crop**: Fit horizontal videos to vertical
- **Thumbnail Selection**: Choose best frame
- **Caption Generator**: AI-powered descriptions
- **Hashtag Research**: Trending tag suggestions
- **Sound Matching**: Find trending audio

### Advanced Features

- **Duet/Stitch Settings**: Enable collaborations
- **Comment Controls**: Filter and moderate
- **Download Permissions**: Allow/disable saves
- **Audience Controls**: Age restrictions
- **Disclosure Labels**: Ad/promotion tags

## TikTok Algorithm

### Key Factors

1. **Completion Rate**: Videos watched to end
2. **Engagement**: Likes, comments, shares
3. **Re-watches**: Multiple views per user
4. **Time Spent**: Total watch time
5. **Creator Activity**: Consistent posting

### For You Page (FYP) Tips

- Hook viewers in first 3 seconds
- Use trending sounds and effects
- Post at peak times
- Engage with comments quickly
- Create niche-specific content

## Best Practices

### Content Strategy

1. **Consistency**: Post 1-4 times daily
2. **Timing**:
   - Morning: 6-9 AM
   - Afternoon: 12-3 PM
   - Evening: 7-11 PM
3. **Length**: 15-30 seconds perform best
4. **Series**: Create episodic content
5. **Trends**: Jump on challenges early

### Hashtag Strategy

- **Quantity**: 3-5 hashtags optimal
- **Mix**: Trending + niche + branded
- **Placement**: In description, not comments
- **Research**: Check hashtag view counts
- **Create**: Start your own challenges

### Engagement Tactics

- Reply to comments with videos
- Go live regularly (1000+ followers)
- Create duettable content
- Ask questions in videos
- Use polls and Q&A features

## Troubleshooting

### Common Issues

**Upload Failed**

- Check video specifications
- Verify internet connection
- Ensure account standing
- Try different file format

**Low Views**

- Improve video hooks
- Post at optimal times
- Use trending elements
- Engage with community

**Account Restrictions**

- Review community guidelines
- Check for violations
- Appeal if necessary
- Build back reputation

**Connection Problems**

- Re-authenticate account
- Check API status
- Verify permissions
- Contact support

### Error Messages

- "Video processing failed": Check format/size
- "Network error": Verify connection
- "Permission denied": Re-grant access
- "Rate limited": Reduce posting frequency
