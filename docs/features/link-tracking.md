# Link Tracking

Track clicks, conversions, and engagement on links shared in your GenFeed content. Understand which content drives traffic and measure ROI.

## Overview

Link Tracking helps you:

- Track clicks on links in your content
- Measure conversion rates
- Understand which content drives traffic
- Calculate ROI on your content
- Optimize link placement and messaging

## How It Works

### GenFeed Link Shortening

When you add links to your content, GenFeed automatically:

1. Creates a shortened, trackable link
2. Tracks clicks and engagement
3. Provides analytics on link performance
4. Measures conversions and ROI

### Link Analytics

Track comprehensive metrics:

- **Total Clicks**: How many times link was clicked
- **Unique Clicks**: Number of unique users who clicked
- **Click-Through Rate (CTR)**: Percentage of viewers who clicked
- **Conversion Rate**: Percentage of clicks that converted
- **Traffic Sources**: Where clicks came from
- **Geographic Data**: Where clicks originated
- **Device Data**: Mobile vs desktop clicks
- **Time Data**: When clicks occurred

## Setting Up Link Tracking

### Adding Links to Content

**In Captions:**

1. Create or edit caption
2. Add link URL
3. GenFeed automatically creates trackable link
4. Link is shortened and tracked

**In Articles:**

1. Create or edit article
2. Add links in content
3. Links automatically tracked
4. Analytics available in Analytics app

**In Social Posts:**

1. Create post in Publisher
2. Add link to post
3. Link automatically tracked
4. View analytics after publishing

### Link Customization

**Custom Short Links:**

- Create branded short links
- Customize link text
- Add UTM parameters
- Track specific campaigns

**UTM Parameters:**

- Automatically added to links
- Track campaign sources
- Measure campaign effectiveness
- Integrate with analytics tools

## ROI Calculation

### Measuring ROI

**Calculate ROI:**

1. Track link clicks
2. Measure conversions
3. Calculate conversion value
4. Compare to content cost

**ROI Formula:**

```
ROI = (Conversion Value - Content Cost) / Content Cost × 100
```
