# Command Palette

The Command Palette (`⌘K` or `Ctrl+K`) is a powerful quick-access tool that lets you navigate, search, and execute actions across GenFeed apps instantly.

## Overview

The Command Palette provides instant access to:

- Navigation to any page or feature
- Quick actions and commands
- Search across your content
- Keyboard shortcuts for power users

## Accessing Command Palette

### Keyboard Shortcut

- **Mac**: `⌘K` (Command + K)
- **Windows/Linux**: `Ctrl+K`

### Visual Access

- Click the search icon in the top navigation bar
- Or use the keyboard shortcut from anywhere in the app

## Features

### Quick Navigation

Navigate to any page or feature instantly:

1. Open Command Palette (`⌘K`)
2. Start typing the page or feature name
3. Select from filtered results
4. Press Enter to navigate

**Examples:**

- Type "studio" → Navigate to Studio app
- Type "analytics" → Navigate to Analytics
- Type "settings" → Open Settings

### Search Content

Search across your content:

1. Open Command Palette (`⌘K`)
2. Type your search query
3. Results show matching content
4. Select to open content

**Searchable Content:**

- Ingredients (captions, images, videos)
- Collections
- Folders
- Published content
- Scheduled posts

### Quick Actions

Execute actions without navigating:

**Common Actions:**

- Create new ingredient
- Create new collection
- Create new folder
- Schedule post
- Generate content

**How to Use:**

1. Open Command Palette (`⌘K`)
2. Type action name (e.g., "create ingredient")
3. Select action
4. Action dialog opens

### Keyboard Navigation

Navigate results with keyboard:

- **Arrow Keys**: Navigate up/down through results
- **Enter**: Select highlighted result
- **Esc**: Close Command Palette
- **Tab**: Cycle through result categories

## Keyboard Shortcuts

### Command Palette Shortcuts

- `⌘K` / `Ctrl+K`: Open Command Palette
- `Esc`: Close Command Palette
- `↑` / `↓`: Navigate results
- `Enter`: Select result
- `Tab`: Cycle categories

### App-Wide Shortcuts

Command Palette works from anywhere:

- Studio: `⌘K` → Navigate or create
- Manager: `⌘K` → Search or navigate
- Publisher: `⌘K` → Schedule or navigate
- Analytics: `⌘K` → Navigate or search
- Automation: `⌘K` → Create workflow or navigate

## Troubleshooting

### Command Palette Not Opening

**Check:**

- Keyboard shortcut conflicts
- Browser focus (click in app first)
- Browser compatibility

**Solution:**

- Try clicking search icon instead
- Check browser console for errors
- Refresh page if needed

### Results Not Showing

**Check:**

- Typo in search term
- Content permissions
- Search indexing

**Solution:**

- Try different search terms
- Check content visibility
- Wait for search indexing

### Slow Performance

**Check:**

- Large content library
- Browser performance
- Network connection

**Solution:**

- Narrow search terms
- Clear browser cache
- Check network connection
