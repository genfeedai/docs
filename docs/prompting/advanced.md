# Advanced Prompting Techniques

Master advanced prompting techniques to get consistently better results from GenFeed. Learn chain of thought prompting, few-shot examples, style consistency, brand voice integration, and workflow prompting.

## Overview

Advanced prompting techniques help you get more consistent, higher-quality results. These techniques build on basic prompting skills and help you create content that matches your brand and goals.

## Chain of Thought Prompting

### What Is Chain of Thought?

Chain of thought prompting breaks complex tasks into logical steps, helping the AI think through the problem systematically.

### How to Use

**Structure:**

```
First, [step 1]
Then, [step 2]
Next, [step 3]
Finally, [step 4]
```

**Example:**

```
Create a product launch video:

First, analyze the target audience for luxury watches.
Then, identify their main pain points and desires.
Next, create a video that addresses these pain points.
Finally, include a clear call-to-action that resonates with this audience.

Video should showcase the watch's premium features while speaking directly to luxury watch enthusiasts' values.
```

### Benefits

- More logical content structure
- Better problem-solving
- Clearer thinking process
- More targeted results

### When to Use

- Complex content creation
- Multi-step processes
- Problem-solving content
- Strategic content planning

## Few-Shot Prompting

### What Is Few-Shot Prompting?

Few-shot prompting provides examples of desired output, helping the AI understand the style and format you want.

### How to Use

**Structure:**

```
Create [content type] in this style:

Example 1: [example 1]
Example 2: [example 2]
Example 3: [example 3]

Now create: [your request]
```

## Style Consistency

### Maintaining Style Across Content

**Create Style Guide:**

1. Define your brand style
2. Document style elements
3. Create style examples
4. Reference in prompts

**Style Elements:**

- Color palette
- Visual style
- Tone of voice
- Content themes
- Format preferences

### Style Prompt Template

```
Create [content] that matches our brand style:

Brand Style:
- Colors: [color palette]
- Visual: [visual style description]
- Tone: [tone of voice]
- Themes: [content themes]
- Format: [format preferences]

Reference: [link to style examples or previous content]
```

### Building Style Library

- Save successful content as style examples
- Document what works
- Create style templates
- Reuse style references
- Build consistent brand

## Brand Voice Integration

### What Is Brand Voice?

Brand voice is the personality and tone your brand uses in all communications. Integrating brand voice into prompts ensures consistent messaging.

### Defining Brand Voice

**Elements:**

- Personality traits (friendly, professional, playful)
- Tone (casual, formal, conversational)
- Values (what your brand stands for)
- Audience (who you're speaking to)
- Examples (previous content that represents your voice)

### Brand Voice Prompt Template

```
Create [content] using our brand voice:

Brand Personality: [personality traits]
Tone: [tone description]
Values: [brand values]
Audience: [target audience]
Voice Examples: [examples of brand voice]

Content should feel authentic to our brand while [content goal].
```

### Example

```
Write a social media post using our brand voice:

Brand Personality: Friendly, approachable, knowledgeable
Tone: Conversational, helpful, encouraging
Values: Sustainability, quality, customer care
Audience: Environmentally conscious consumers
Voice Examples: [reference to previous posts]

Post should introduce our new eco-friendly product line while maintaining our warm, helpful tone.
```

## Workflow Prompting

### What Is Workflow Prompting?

Workflow prompting creates prompts that work together in a sequence, building on each other to create complete content pipelines.

### Creating Workflows

**Step-by-Step Approach:**

1. **Define Workflow Goal**: What's the end result?
2. **Break Into Steps**: What steps are needed?
3. **Create Step Prompts**: Prompt for each step
4. **Connect Steps**: How do steps relate?
5. **Test Workflow**: Test entire workflow

### Workflow Example

**Content Series Workflow:**

```
Step 1 - Generate Script:
Write a 60-second video script about [topic].
Structure: Hook (5s), main content (50s), CTA (5s).
Tone: [brand tone]
Include: [key points]

Step 2 - Generate Voiceover:
Convert script to professional voiceover.
Voice: [voice style]
Tone: [emotional tone]
Speed: [pace]

Step 3 - Generate Video:
Create video matching voiceover.
Scene: [scene description]
Style: [visual style]
Duration: 60 seconds
Sync: Match video to voiceover timing

Step 4 - Generate Captions:
Create captions for video.
Style: [caption style]
Timing: Synchronized with voiceover
```

### Workflow Best Practices

- **Start with End Goal**: Know what you want to achieve
- **Plan Steps**: Think through entire workflow
- **Test Each Step**: Ensure each step works
- **Iterate**: Refine workflow based on results
- **Document**: Save successful workflows

## Prompt Optimization

### Iterative Refinement

**Process:**

1. **Initial Prompt**: Start with basic prompt
2. **Generate Content**: Create first version
3. **Evaluate Results**: What works? What doesn't?
4. **Refine Prompt**: Improve based on results
5. **Regenerate**: Create improved version
6. **Repeat**: Continue refining

### Prompt Refinement Checklist

- [ ] More specific details added
- [ ] Style references included
- [ ] Brand voice integrated
- [ ] Examples provided
- [ ] Structure defined
- [ ] Quality expectations set

## Common Advanced Patterns

### Pattern 1: Context + Style + Goal

```
Context: [background information]
Style: [style reference or description]
Goal: [what you want to achieve]
Create: [content request]
```

### Pattern 2: Problem + Solution + Format

```
Problem: [problem to solve]
Solution Approach: [how to solve it]
Format: [desired format]
Create: [content that solves problem]
```

### Pattern 3: Audience + Message + Style

```
Audience: [who you're speaking to]
Message: [what you want to communicate]
Style: [how to communicate it]
Create: [content matching all elements]
```

## Tips for Advanced Prompting

### General Tips

1. **Be Specific**: More detail = better results
2. **Use Examples**: Show what you want
3. **Define Context**: Provide background
4. **Set Expectations**: Clear quality goals
5. **Iterate**: Refine based on results

### Technique-Specific Tips

**Chain of Thought:**

- Break complex tasks into logical steps
- Think through the process
- Guide AI thinking

**Few-Shot:**

- Provide 2-3 good examples
- Examples should be similar in style
- Clear what makes examples good

**Style Consistency:**

- Document your style
- Reference previous successful content
- Build style library

**Brand Voice:**

- Define voice clearly
- Provide voice examples
- Maintain consistency

**Workflow:**

- Plan entire workflow
- Test each step
- Document successful workflows

## Troubleshooting Advanced Prompts

### Results Don't Match Style

- Provide more style examples
- Be more specific about style elements
- Reference successful previous content
- Refine style description

### Brand Voice Not Consistent

- Provide more voice examples
- Be clearer about personality traits
- Include tone guidelines
- Reference brand voice documentation

### Workflow Not Working

- Test each step individually
- Verify step connections
- Check step order
- Simplify workflow if needed
