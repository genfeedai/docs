# Content Type Prompting Guide

Different content types require different prompting approaches. Learn how to optimize your prompts for videos, images, articles, voice, music, and captions.

## Overview

Each content type has unique characteristics and requirements. Understanding how to prompt for each type will help you get better results and create more effective content.

## Video Prompting

### Video Prompt Structure

**Essential Elements:**

1. **Scene Description**: What's happening visually
2. **Camera Movement**: How camera behaves
3. **Subject Action**: What moves in scene
4. **Style/Mood**: Visual treatment
5. **Duration**: Length and pacing
6. **Transitions**: How scene evolves

### Video Prompt Template

```
Create a [duration]-second video of [scene description].
Camera: [camera movement and angles]
Action: [what happens in scene]
Style: [visual style and mood]
Lighting: [lighting conditions]
Transitions: [transition style]
Music: [music style and mood]
```

### Video Prompting Best Practices

- **Be Specific**: Describe scenes in detail
- **Include Motion**: Specify camera and subject movement
- **Define Style**: Clear style and mood direction
- **Set Duration**: Specify video length
- **Describe Transitions**: How scenes connect

### Video Examples

**Product Showcase:**

```
Create a 30-second product reveal video.
Scene: Sleek smartphone rotating on pedestal.
Camera: Slow 360-degree rotation, close-up details.
Action: Features highlight with smooth animations.
Style: Modern, minimalist, Apple-inspired.
Lighting: Bright studio lighting with reflections.
Transitions: Smooth cuts every 3-5 seconds.
Music: Futuristic electronic beat building to climax.
```

**Social Media Content:**

```
Create a 15-second TikTok-style video.
Scene: Transformation from before to after state.
Camera: Fast-paced cuts, trendy transitions.
Action: Quick transformation with motion blur.
Style: Bright, high contrast, saturated colors.
Transitions: Trendy spin transition.
Music: Current trending audio clip.
Duration: Exactly 15 seconds, vertical 9:16 aspect ratio.
```

## Image Prompting

### Image Prompt Structure

**Essential Elements:**

1. **Subject**: Main focus of image
2. **Style/Medium**: Artistic approach
3. **Environment**: Setting and background
4. **Lighting/Mood**: Atmosphere
5. **Composition**: Camera angle and framing
6. **Details**: Specific elements

### Image Prompt Template

```
[Subject description] in [environment/setting].
Style: [artistic style or photography type]
Lighting: [lighting conditions and mood]
Composition: [camera angle and framing]
Details: [specific elements and features]
Mood: [emotional tone]
```

### Image Prompting Best Practices

- **Describe Subject Clearly**: Main focus should be clear
- **Specify Style**: Include artistic or photography style
- **Include Lighting**: Lighting dramatically affects mood
- **Define Composition**: Camera angle and framing matter
- **Add Details**: Specific details improve results

### Image Examples

**Product Photography:**

```
Minimalist product shot of luxury watch.
Style: Commercial product photography.
Background: Pure white seamless backdrop.
Lighting: Dramatic side lighting creating metallic reflections.
Angle: 45-degree angle emphasizing craftsmanship.
Details: Sharp focus on watch face, visible texture of materials.
Mood: Premium, sophisticated, high-end.
```

**Artistic Image:**

```
Cyberpunk cityscape at night.
Style: Digital matte painting, neon aesthetic.
Environment: Futuristic city with towering buildings.
Lighting: Neon signs, rain-slicked streets reflecting lights.
Composition: Aerial perspective showing city expanse.
Details: Flying vehicles, holographic advertisements, rain effects.
Mood: Dystopian, vibrant, high-energy.
```

## Article Prompting

### Article Prompt Structure

**Essential Elements:**

1. **Topic**: What article is about
2. **Target Audience**: Who it's for
3. **Word Count**: Desired length
4. **Tone**: Writing style
5. **Structure**: Article organization
6. **SEO Keywords**: Search optimization

### Article Prompt Template

```
Write a [word count]-word article about [topic].
Target audience: [describe audience]
Tone: [professional/casual/friendly/authoritative]
Structure: [introduction, X main points, conclusion]
SEO keywords: [keyword1, keyword2, keyword3]
Include: [specific requirements]
Exclude: [what to avoid]
```

### Article Prompting Best Practices

- **Define Audience**: Know who you're writing for
- **Specify Length**: Word count helps structure content
- **Set Tone**: Match tone to audience and purpose
- **Request Structure**: Ask for specific sections
- **Include Keywords**: Naturally integrate SEO keywords

### Article Examples

**Blog Post:**

```
Write a 1,500-word blog post about email marketing strategies.
Target audience: Small business owners new to email marketing.
Tone: Friendly, practical, encouraging.
Structure: Introduction, 5 main strategies with examples, conclusion with CTA.
SEO keywords: email marketing, small business, marketing automation, email campaigns.
Include: Real-world examples, actionable tips, statistics.
Exclude: Technical jargon, overly complex strategies.
```

**SEO Article:**

```
Write a 2,000-word SEO-optimized article about remote work productivity.
Target audience: Remote workers and managers.
Tone: Professional yet accessible.
Structure: Introduction with hook, 7 productivity tips with subheadings, conclusion.
SEO keywords: remote work, productivity tips, work from home, remote team management.
Include: Statistics, expert quotes, practical tools.
Optimize for: Search engines while maintaining readability.
```

## Voice Prompting

### Voice Prompt Structure

**Essential Elements:**

1. **Text Content**: What to say
2. **Voice Style**: Voice characteristics
3. **Emotion/Tone**: Emotional delivery
4. **Speed**: Speaking pace
5. **Pauses**: Natural pauses

### Voice Prompt Template

```
Convert this text to speech:
[Text content]

Voice: [voice style - professional, friendly, authoritative]
Tone: [emotional tone - enthusiastic, calm, serious]
Speed: [speaking pace - normal, slow, fast]
Pauses: [natural pauses at punctuation]
Emphasis: [words or phrases to emphasize]
```

### Voice Prompting Best Practices

- **Use Punctuation**: Punctuation creates natural pauses
- **Break Long Text**: Split into sentences
- **Match Voice to Content**: Professional content = professional voice
- **Specify Emotion**: Emotional tone affects delivery
- **Test Different Voices**: Try various voices for your brand

### Voice Examples

**Professional Narration:**

```
Convert to professional voiceover:
"Welcome to our product launch. Today we're introducing revolutionary technology that will transform how you work."

Voice: Professional, authoritative, clear.
Tone: Confident and engaging.
Speed: Moderate pace, clear enunciation.
Pauses: Natural pauses at commas and periods.
```

**Friendly Explanation:**

```
Convert to friendly, conversational voice:
"Hey there! Let me show you how this feature works. It's really simple once you get the hang of it."

Voice: Friendly, warm, approachable.
Tone: Enthusiastic and helpful.
Speed: Slightly faster, conversational pace.
Pauses: Natural conversational pauses.
```

## Music Prompting

### Music Prompt Structure

**Essential Elements:**

1. **Genre**: Music style
2. **Mood**: Emotional feel
3. **Tempo**: Speed (BPM)
4. **Instruments**: What instruments to use
5. **Duration**: Length of track

### Music Prompt Template

```
Generate [duration]-second [genre] music track.
Mood: [emotional feel - upbeat, calm, energetic]
Tempo: [BPM or speed description]
Instruments: [specific instruments]
Style: [musical style characteristics]
Energy: [energy level - low, medium, high]
```

### Music Prompting Best Practices

- **Be Specific About Genre**: Clear genre helps
- **Describe Mood**: Emotional feel is important
- **Include Tempo**: BPM or speed description
- **Specify Instruments**: If instruments matter
- **Match to Content**: Music should complement content

### Music Examples

**Background Music:**

```
Generate 60-second ambient background music.
Genre: Ambient, electronic.
Mood: Calm, focused, unobtrusive.
Tempo: Slow, around 70 BPM.
Instruments: Soft synthesizers, subtle pads.
Style: Minimal, atmospheric, background-friendly.
Energy: Low to medium, non-distracting.
```

**Upbeat Track:**

```
Generate 30-second upbeat electronic track.
Genre: Electronic, house.
Mood: Energetic, positive, motivating.
Tempo: Fast, around 128 BPM.
Instruments: Synthesizers, drums, bass.
Style: Modern, energetic, danceable.
Energy: High, attention-grabbing.
```

## Caption Prompting

### Caption Prompt Structure

**Essential Elements:**

1. **Video Content**: What video shows
2. **Tone**: Caption tone
3. **Length**: Caption length
4. **Style**: Caption style
5. **Timing**: When captions appear

### Caption Prompt Template

```
Generate captions for video about [topic].
Video content: [describe what video shows]
Tone: [caption tone - professional, casual, fun]
Length: [short/medium/long captions]
Style: [caption style - simple, animated, styled]
Timing: [when captions appear - beginning, throughout, key moments]
```

### Caption Prompting Best Practices

- **Describe Video Content**: Help AI understand context
- **Match Tone**: Caption tone should match video
- **Specify Length**: Short for quick info, longer for explanations
- **Define Style**: Simple or styled captions
- **Consider Timing**: When captions appear matters

### Caption Examples

**Simple Captions:**

```
Generate simple captions for product demo video.
Video content: Showcasing smartphone features and capabilities.
Tone: Professional, clear, informative.
Length: Short captions highlighting key features.
Style: Simple white text, easy to read.
Timing: Appear at key feature moments.
```

**Styled Captions:**

```
Generate styled captions for social media video.
Video content: Quick tips and tricks tutorial.
Tone: Fun, energetic, engaging.
Length: Medium captions with key points.
Style: Bold, animated, colorful, matches brand.
Timing: Throughout video, synchronized with speech.
```

## Content Type Comparison

### Prompting Complexity

- **Videos**: Most complex (scene, camera, motion, style)
- **Images**: Medium complexity (subject, style, composition)
- **Articles**: Medium complexity (topic, audience, structure)
- **Voice**: Lower complexity (text, voice style, tone)
- **Music**: Lower complexity (genre, mood, tempo)
- **Captions**: Lower complexity (content, tone, style)

### Common Elements

All content types benefit from:

- **Specificity**: More detail = better results
- **Style Definition**: Clear style direction
- **Target Audience**: Know your audience
- **Purpose**: Define content purpose
- **Quality Expectations**: Set quality standards

## Tips for Better Results

### Across All Content Types

1. **Be Specific**: More detail improves results
2. **Use Examples**: Reference styles you like
3. **Iterate**: Refine prompts based on results
4. **Match Model**: Choose right model for content type
5. **Test Variations**: Try different approaches

### Content Type-Specific

**Videos:**

- Focus on motion and transitions
- Describe camera movement
- Specify duration and pacing

**Images:**

- Include composition details
- Specify lighting conditions
- Describe style clearly

**Articles:**

- Define audience clearly
- Request specific structure
- Include SEO naturally

**Voice:**

- Use proper punctuation
- Match voice to content tone
- Specify emotional delivery

**Music:**

- Be clear about genre
- Describe mood accurately
- Match tempo to content
