# Keyboard Shortcuts

Master GenFeed.ai's keyboard shortcuts to significantly speed up your content creation workflow. These shortcuts work across all major browsers and operating systems.

## Custom Shortcuts

### Creating Custom Shortcuts

1. Navigate to **Settings** → **Keyboard Shortcuts**
2. Click **"Add Custom Shortcut"**
3. Choose the action from dropdown
4. Press your desired key combination
5. Click **"Save"**

### Popular Custom Shortcuts

Users often create shortcuts for:
- Frequently used prompts
- Specific model switches
- Export format preferences
- Template insertions
- Team collaboration actions

### Shortcut Conflicts

If a shortcut conflicts with your system:
1. GenFeed.ai will notify you
2. Choose to override or select different keys
3. System shortcuts take precedence in some cases

## Platform-Specific Shortcuts

### Browser-Specific

**Chrome/Edge Additional Shortcuts**

- Web Console: `Ctrl + Shift + K` / `Cmd + Alt + K`

### Mobile Gestures

**iOS (iPad with Keyboard)**
- Command palette: `Cmd + K`

**Android (with Keyboard)**
- Most desktop shortcuts work

## Productivity Tips

### Shortcut Combinations

Chain shortcuts for maximum efficiency:

1. `Ctrl/Cmd + N` → New document
2. `Ctrl/Cmd + P` → Select prompt template
3. `Ctrl/Cmd + 4` → Switch to GPT-4
4. `Ctrl/Cmd + Enter` → Generate
5. `Ctrl/Cmd + E` → Export

### Context-Aware Shortcuts

Shortcuts change based on context:
- In editor: Text formatting shortcuts active
- In prompt field: Generation shortcuts active
- In settings: Navigation shortcuts active

### Learning Shortcuts

1. **Start with essentials**: Learn 5 most-used shortcuts first
2. **Practice daily**: Use shortcuts instead of mouse
3. **Gradual expansion**: Add 2-3 new shortcuts weekly
4. **Shortcut hints**: Enable tooltips in settings

## Troubleshooting

### Shortcuts Not Working?

1. **Check browser focus**: Click inside GenFeed.ai window
2. **Disable extensions**: Some extensions override shortcuts
3. **Update browser**: Ensure latest version
4. **Clear cache**: Ctrl/Cmd + Shift + Delete
5. **Check permissions**: Some shortcuts need specific permissions

### Reset Shortcuts

To reset all shortcuts to defaults:
1. Go to Settings → Keyboard Shortcuts
2. Click "Reset to Defaults"
3. Confirm action
4. Refresh page

### Shortcut Conflicts

Common conflicts and solutions:
- **Ctrl/Cmd + K**: May conflict with browser search
- **Ctrl/Cmd + L**: Browser address bar vs. insert link
- **F1-F12**: System functions vs. app shortcuts
