## Quick Start

### Core (Self-Hosted)

```bash
git clone https://github.com/genfeedai/core.git
cd core
bun install
cp .env.example .env.local  # Add your API keys
bun dev
```

[**Full Installation Guide →**](/core/installation)

### Cloud

1. [Sign up at genfeed.ai](https://genfeed.ai/signup)
2. Create your organization
3. Set up your brand (logo, voice, tone)
4. Start generating content
