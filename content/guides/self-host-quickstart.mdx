# Self-Host GenFeed in 5 Minutes

Get GenFeed Core running on your own infrastructure. Generate AI content with full control over models, data, and costs.

<Callout type="info">
**Just want to use GenFeed?** [GenFeed Cloud](https://genfeed.ai) is the fastest way to start — no setup required, all features included, free tier available.
</Callout>

## Prerequisites

- Node.js 18+ (or Bun)
- MongoDB 7+
- Redis 7+
- At least one AI provider API key (Replicate, fal.ai, or OpenAI)

## Quick Start

<Steps>

### Clone the repository

```bash
git clone https://github.com/genfeedai/core.git
cd core
```

### Install dependencies

<Tabs items={['Bun (Recommended)', 'npm']}>
  <Tabs.Tab>
```bash
bun install
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
npm install
```
  </Tabs.Tab>
</Tabs>

### Configure environment

```bash
cp .env.example .env.local
```

Edit `.env.local` with your settings:

```bash
# Required
MONGODB_URI=mongodb://localhost:27017/genfeed
REDIS_URL=redis://localhost:6379

# AI Providers (at least one required)
REPLICATE_API_TOKEN=your_token_here
FAL_KEY=your_key_here

# Optional
PORT=3001
NODE_ENV=development
```

### Start the server

<Tabs items={['Bun', 'npm']}>
  <Tabs.Tab>
```bash
bun run dev
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
npm run dev
```
  </Tabs.Tab>
</Tabs>

### Generate your first image

```bash
curl -X POST http://localhost:3001/api/generate/image \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A modern tech startup office, minimalist design",
    "model": "flux-schnell",
    "width": 1024,
    "height": 1024
  }'
```

</Steps>

## Docker Compose

For a production-ready setup with all dependencies:

```yaml
# docker-compose.yml
version: '3.8'

services:
  genfeed:
    build: .
    ports:
      - "3001:3001"
    environment:
      - MONGODB_URI=mongodb://mongo:27017/genfeed
      - REDIS_URL=redis://redis:6379
      - REPLICATE_API_TOKEN=${REPLICATE_API_TOKEN}
    depends_on:
      - mongo
      - redis

  mongo:
    image: mongo:7
    volumes:
      - mongo_data:/data/db
    ports:
      - "27017:27017"

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

volumes:
  mongo_data:
```

```bash
docker compose up -d
```

## What's Included

- Brand kit management
