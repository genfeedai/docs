# Voice Recognition Setup

Genfeed.ai's voice recognition feature allows you to create content hands-free by speaking your ideas directly into the platform. This guide will help you set up and optimize voice recognition for the best results.

## System Requirements

### Desktop Requirements

- **Operating System**: Windows 10+, macOS 10.14+, Linux (Ubuntu 20.04+)
- **Browser**: Chrome 90+, Edge 90+, Safari 14+, Firefox 88+
- **Microphone**: Any USB or built-in microphone
- **Internet**: Stable connection (minimum 1 Mbps)

### Mobile Requirements

- **iOS**: Version 14.0 or higher
- **Android**: Version 8.0 or higher
- **Genfeed.ai Mobile App**: Latest version

## Enabling Voice Recognition

### Step 1: Check Microphone Permissions

**Browser Permissions**

1. Click the lock icon in your browser's address bar
2. Find "Microphone" in the permissions list
3. Select "Allow" for docs.genfeed.ai
4. Refresh the page

**System Permissions (Windows)**

1. Go to Settings → Privacy → Microphone
2. Enable "Allow apps to access your microphone"
3. Ensure your browser is allowed

**System Permissions (macOS)**

1. Go to System Preferences → Security & Privacy → Privacy
2. Select Microphone from the left panel
3. Check the box next to your browser

### Step 2: Enable Voice Recognition in Genfeed.ai

1. Log in to your Genfeed.ai account
2. Navigate to **Settings** → **Voice Recognition**
3. Toggle **"Enable Voice Input"** to ON
4. Select your preferred language
5. Choose voice activation method:
   - **Push-to-Talk**: Hold button to record
   - **Voice Activation**: Automatic detection
   - **Always Listening**: Continuous mode (Pro plans)

### Step 3: Configure Audio Settings

1. Click **"Audio Settings"** in Voice Recognition menu
2. Select your microphone from the dropdown
3. Adjust input sensitivity using the slider
4. Test your microphone with the **"Test Audio"** button
5. Enable noise cancellation (recommended)

## Voice Commands

### Basic Commands

- **"Start recording"**: Begin voice input
- **"Stop recording"**: End voice input
- **"New paragraph"**: Start a new paragraph
- **"New line"**: Insert line break
- **"Delete last sentence"**: Remove previous sentence
- **"Clear all"**: Delete all dictated text

### Punctuation Commands

- **"Period"** or **"Full stop"**: .
- **"Comma"**: ,
- **"Question mark"**: ?
- **"Exclamation mark"**: !
- **"Colon"**: :
- **"Semicolon"**: ;
- **"Open quotes"** / **"Close quotes"**: " "
- **"Open parenthesis"** / **"Close parenthesis"**: ( )

### Formatting Commands

- **"Bold [text]"**: Makes text bold
- **"Italic [text]"**: Makes text italic
- **"Underline [text]"**: Underlines text
- **"Heading 1/2/3"**: Creates headings
- **"Bullet point"**: Starts bulleted list
- **"Number list"**: Starts numbered list

### AI Commands

- **"Generate content"**: Process dictated text with AI
- **"Improve this"**: Enhance the dictated text
- **"Make it longer"**: Expand the content
- **"Make it shorter"**: Condense the content
- **"Change tone to [formal/casual/friendly]"**: Adjust writing style

## Advanced Voice Features

### Custom Voice Commands

Create your own voice shortcuts:

1. Go to **Settings** → **Voice Recognition** → **Custom Commands**
2. Click **"Add New Command"**
3. Set the voice trigger phrase
4. Define the action or text replacement
5. Save and test

Example custom commands:

- **"My signature"** → Inserts your email signature
- **"Company name"** → Types "Genfeed.ai"
- **"Current date"** → Inserts today's date

### Voice Templates

Speed up content creation with voice-activated templates:

1. Navigate to **Templates** → **Voice Templates**
2. Select or create a template
3. Assign a voice command
4. Use the command during dictation

Example: Say **"Blog intro template"** to insert your standard blog introduction format.

## Optimizing Voice Recognition Accuracy

### Environmental Setup

1. **Quiet Environment**
   - Minimize background noise
   - Close windows and doors
   - Turn off fans or AC if possible

2. **Microphone Placement**
   - Position 6-12 inches from mouth
   - Keep at chest level
   - Avoid breathing directly into mic

3. **Acoustic Treatment**
   - Use in carpeted rooms when possible
   - Avoid echo-prone spaces
   - Consider a pop filter for professional use

### Speaking Tips

1. **Clear Articulation**
   - Speak naturally but clearly
   - Don't rush or mumble
   - Maintain consistent volume

2. **Proper Pacing**
   - Speak at moderate speed
   - Pause between sentences
   - Allow system to process

3. **Command Clarity**
   - Slight pause before commands
   - Emphasize command words
   - Use consistent command phrases

### Troubleshooting Common Issues

**Low Recognition Accuracy**

- Check microphone quality
- Adjust input sensitivity
- Run voice training module
- Update browser/app

**Commands Not Working**

- Verify command syntax
- Check language settings
- Ensure proper pausing
- Review custom commands

**Audio Cutting Out**

- Test internet connection
- Check browser permissions
- Disable battery optimization
- Update audio drivers

3. Read provided phrases clearly
