# Studio

The creative workspace where content comes to life.

## Overview

Studio is the main interface for creating content in Genfeed Cloud. It combines the workflow engine with your brand settings to produce on-brand content.

## Interface

### Workspace

- **Projects** — Organize content by campaign or client
- **Assets** — Your generated images, videos, and audio
- **Templates** — Pre-built starting points

### Generation Panel

- **Prompt input** — Describe what you want
- **Model selector** — Choose AI model
- **Brand context** — Auto-applies your voice and tone
- **Output settings** — Format, size, quality

### Asset Library

All generated content is saved to your asset library:

- Filter by type (image, video, audio, text)
- Search by prompt or tags
- Download or publish directly

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Quick actions |
| `Cmd/Ctrl + N` | New generation |
| `Cmd/Ctrl + Enter` | Generate |
| `Cmd/Ctrl + S` | Save to assets |
