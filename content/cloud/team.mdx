# Team Management

Collaborate with your team using roles and permissions.

## Inviting Members

1. Go to **Organization → Team**
2. Click **Invite Member**
3. Enter email address
4. Select role
5. Send invitation

## Roles

| Role | Permissions |
|------|-------------|
| **Owner** | Full access, billing, delete org |
| **Admin** | Manage team, brands, settings |
| **Editor** | Create and edit content |
| **Viewer** | View only, no editing |

## Role Permissions

### Owner
- Everything an Admin can do
- Manage billing and subscription
- Delete organization
- Transfer ownership

### Admin
- Invite and remove members
- Manage brands and settings
- Create and edit all content
- View analytics

### Editor
- Create new content
- Edit own content
- Use all generation features
- Cannot manage team or settings

### Viewer
- View all content
- Download assets
- Cannot create or edit

## Managing Members

### Change Role

1. Go to **Team**
2. Click member's current role
3. Select new role

### Remove Member

1. Go to **Team**
2. Click **...** next to member
3. Select **Remove**
