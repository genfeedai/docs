# Account Settings

Manage your profile, security, and preferences.

## Profile

### Personal Information

- Display name
- Email address
- Avatar

### Notifications

Control which notifications you receive:

- Email notifications
- In-app notifications
- Weekly digest

## Security

### Password

Change your password:

1. Go to **Settings → Security**
2. Enter current password
3. Enter new password
4. Confirm

### Two-Factor Authentication

Enable 2FA for extra security:

1. Go to **Settings → Security**
2. Click **Enable 2FA**
3. Scan QR code with authenticator app
4. Enter verification code

### Sessions

View and manage active sessions:

- See all logged-in devices
- Revoke access to specific sessions
- Log out everywhere

## Connected Accounts

### Social Platforms

Connect accounts for publishing:

- YouTube

### API Keys

For Core features within Cloud:

- Add your own API keys
- Override default models
- Use custom endpoints

## Preferences

### Theme

- Light mode
- Dark mode
- System default

### Language

- English (default)
- More coming soon

## Data & Privacy

### Export Data

Download all your data:

1. Go to **Settings → Data**
2. Click **Export**
3. Receive download link via email

### Delete Account

To delete your account and all data:

1. Go to **Settings → Data**
2. Click **Delete Account**
3. Confirm by typing "DELETE"

⚠️ This action is irreversible.
