## What is Genfeed.ai?

Genfeed.ai is an AI content generation platform designed for businesses. Whether you need marketing videos, social media content, or automated publishing workflows, Genfeed streamlines your content creation process.

### Key Features

- **Multi-Modal Generation** — Images, videos, voice, music, and text
- **40+ AI Models** — GPT-4, Claude, Gemini, Kling, Veo, Leonardo, and more
- **Social Publishing** — Direct publishing to YouTube, TikTok, Instagram, LinkedIn
- **Workflow Automation** — Schedule and automate content pipelines
- **Team Collaboration** — Manage seats, permissions, and brand assets
- **Voice Recognition** — Create content hands-free

## Quick Start

### For Cloud Users

```
1. Sign up at genfeed.ai
2. Create your organization
3. Connect social accounts
4. Start generating content
```

[**Get Started with Cloud →**](/getting-started)

### For Self-Hosted Users

```bash
git clone https://github.com/genfeedai/genfeed-core.git
cd genfeed-core && bun install
cp .env.example .env.local
bun run start:dev
```

[**Full Installation Guide →**](/deployment/self-hosted)
